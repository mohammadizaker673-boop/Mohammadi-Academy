# Quick Reference: Arabic Learning Platform Features

## 🎯 FIVE CORE LEARNING MODULES

### 1️⃣ LESSONS
- Structured curriculum A1-C2 levels
- Vocabulary, grammar, writing systems
- Progressive unlocking based on performance
- Review completed lessons anytime

### 2️⃣ PRACTICE EXERCISES  
- **Multiple choice** - Select correct answers
- **Fill-in-the-blank** - Complete sentences
- **Translation** - Arabic ↔ English
- Instant feedback after each answer
- 70% required to pass
- Detailed explanations for wrong answers

### 3️⃣ PRONUNCIATION TRAINER
- **Letter practice** - 28 Arabic letters
  - Articulation points explanation
  - Slow audio playback (0.6x speed)
  - Voice recording with accuracy scoring
  
- **Word practice** - Common phrases
  - Listen & repeat exercises
  - Phonetic transcription
  - Real-time recording feedback
  - Pronunciation tips

- **Voice Recognition** - Web Speech API enabled
  - Instant accuracy scoring (60-100%)
  - Unlimited recording attempts
  - Detailed feedback on pronunciation

### 4️⃣ ASSESSMENT CENTER
- **Placement Test** (10 questions)
  - Determine proficiency level A1-C1
  - Mixed question types
  - Automatic scoring

- **Practice Tests** (7+ questions)
  - Comprehensive coverage
  - All difficulty levels
  - Detailed performance breakdown

- **Features**:
  - Progress bar for each test
  - Time tracking
  - Results analytics
  - Weak area identification
  - Pass/Fail feedback

### 5️⃣ AI LANGUAGE TUTOR
- **4 Learning Modes**:
  1. **Teaching** - Learn vocabulary, grammar, culture
  2. **Translation** - Multi-language translation
  3. **Writing** - Get composition feedback
  4. **Conversation** - Practice dialogues

- **Languages Supported**:
  - Arabic ↔ English
  - Arabic ↔ Persian
  - English ↔ Persian

- **Features**:
  - Voice input (click microphone)
  - Detailed explanations
  - Transliteration support
  - Pronunciation guidance
  - Copy-to-clipboard
  - Text-to-speech

## 🆘 COMMON QUESTIONS

**Q: Do I need a microphone?**
A: Only for Pronunciation practice. Other features work without.

**Q: Is my progress saved?**
A: Yes! Automatically saved to your browser.

**Q: Can I use this offline?**
A: Mostly yes, except for AI Tutor (needs internet).

**Q: What if I'm stuck?**
A: Use AI Tutor to ask questions anytime!

**Q: How long to reach fluency?**
A: Depends on level and practice frequency. Estimate: 
- A1→A2: 2-3 months
- A2→B1: 3-4 months  
- B1→B2: 4-5 months

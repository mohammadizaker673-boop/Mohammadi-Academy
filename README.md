# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1E3fyewf7BnRAAD07Nbp2D2KhFwOmLvy5

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `VITE_OPENROUTER_API_KEY` in `.env.local` to your OpenRouter API key.
3. Optional: set `VITE_OPENROUTER_MODEL` in `.env.local` if you want a model other than the default `openai/gpt-4o-mini`.
4. Run the app:
   `npm run dev`
